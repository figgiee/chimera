## Tech Debt (Non-Critical)

### 1. Unreachable `task_failed` display path
**Phases 3/5** — TaskChecklist.svelte has XCircle branch for `status === 'failed'`, but chimera-orchestrator.js only emits `task_start`/`task_done`. The rendering code is correct but unreachable until the orchestrator adds failure events.

### 2. Orphaned `/api/sessions/:id/stats` endpoint
**Phase 1** — Backend implements the endpoint but no frontend component calls it.

### 3. RAG direct calls bypass backend proxy
**Phase 4** — api.ts calls `http://localhost:8080` directly for document operations instead of routing through chimera-chat.js. Cross-origin requests depend on RAG server CORS configuration.

### 4. Orphaned exports
**Phases 2/4** — `SendMessageOptions` type and `cn()` utility exported but never imported.

### 5. Synapse cancel UX
**Phase 3** — SYN-04 satisfied via global stop button, but SynapsePanel has no in-panel cancel affordance.

**Total: 5 items across 3 phases + 1 cross-phase**
