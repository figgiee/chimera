<task type="auto">
  <name>Task 2: Add static file serving and SPA fallback</name>
  <files>chimera-chat.js</files>
  <action>
Add static file serving to chimera-chat.js. This replaces the current 404 fallback at the end of handleRequest().

1. Add requires at the top of the file (after existing requires):
```javascript
const path = require('node:path');
const fs = require('node:fs');
```

2. Add constants after the requires:
```javascript
const STATIC_DIR = path.resolve(__dirname, 'web/build');

const MIME_TYPES = {
  '.html': 'text/html; charset=utf-8',
  '.js':   'application/javascript',
  '.mjs':  'application/javascript',
  '.css':  'text/css',
  '.png':  'image/png',
  '.jpg':  'image/jpeg',
  '.svg':  'image/svg+xml',
  '.ico':  'image/x-icon',
  '.json': 'application/json',
  '.woff2': 'font/woff2',
  '.woff':  'font/woff',
  '.txt':  'text/plain',
};
```

3. Add a serveStatic function:
- Takes (req, res, pathname) as arguments
- Resolves `/` to `/index.html`
- Resolves file path using `path.resolve(STATIC_DIR, '.' + urlPath)`
- Guards against path traversal: if resolved path does not startsWith(STATIC_DIR), return 403
- Uses fs.stat to check if file exists and is a file
  - If yes: writeHead(200, Content-Type from MIME_TYPES) THEN pipe(res). writeHead MUST come BEFORE pipe.
  - If no: serve SPA fallback — writeHead(200, text/html) THEN pipe 200.html. writeHead MUST come BEFORE pipe.

4. Replace the final 404 sendJson block in handleRequest() with a call to serveStatic(req, res, url.pathname). This means: after all /api/* route checks, if nothing matched, serve static files or SPA fallback.

CRITICAL: writeHead() must be called BEFORE fs.createReadStream().pipe(res). Calling it after causes ERR_HTTP_HEADERS_SENT.

Do NOT add cache headers, etag, or gzip — those are out of scope for Phase 1.
  </action>

- STATIC_DIR points to web/build
- MIME_TYPES object exists with common extensions
- serveStatic function exists with path traversal guard
- writeHead is called BEFORE pipe in both the static file and SPA fallback code paths
- The old 404 sendJson at the end of handleRequest is replaced with serveStatic call
  </verify>
